# Rider Road

## 機能の紹介

# 主な機能

* バイク乗りのためのおすすめツーリングスポットを紹介するアプリです。

* 全国のツーリングスポットを写真と地図で見ることができます。

## 使用技術

# 技術のこだわりポイント

* インフラにAWSを使用

* Google Mapを埋め込むことで場所をわかりやすく表示

* UI/UXにこだわって、見やすさの工夫をした
